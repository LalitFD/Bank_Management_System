# 💰 Bank Management System

This is a simple **Bank Management System** developed using **Java (NetBeans IDE)** and **MySQL** for backend database.

## 🚀 Features

- Create New Account
- Deposit Money
- Withdraw Money
- Check Balance
- Delete Account
- View All Customers

## 🛠️ Tech Stack

- Java (Core Java)
- NetBeans IDE
- MySQL Database
- JDBC (Java Database Connectivity)
